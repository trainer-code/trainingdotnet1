Instructions

  1. Download the battleship zip package from the link above
  2. Unzip the package using WinZip/7-Zip/tar or similar software
  3. Open the extracted directory. You need to have a bin directory

Instruction for Linux

  1. Run following in a terminal ./bin/Release/netcoreapp3.1/linux-x64/publish/Battleship.Ascii
  2. If you do not have linux, you can use docker to run the linux version. In a terminal run following:
  3. docker run -it -v ${PWD}:/battleship mcr.microsoft.com/dotnet/core/sdk:3.1 bash
  4. then run following command inside the docker container:
  5. ./bin/Release/netcoreapp3.1/linux-x64/publish/Battleship.Ascii

Instruction for Windows (NOT READY)

  1. Double click on ./bin/Release/netcoreapp3.1/win-x64/publish/Battleship.Ascii.exe

Instruction for Mac (NOT READY)

  1. Run following in a terminal ./bin/Release/netcoreapp3.1/osx-x64/publish/Battleship.Ascii
